## How to Contribute

This repository is used to practice forking, cloning, committing, and creating pull requests.  
Follow the steps exactly.

### Step 1: Fork the Repository
Click the Fork button on the top-right of this repository to create your own copy.

### Step 2: Clone Your Fork
Clone your fork, not the original repository:

```bash
git clone https://github.com/YOUR-USERNAME/GitHub-Demo.git
cd GitHub-Demo
````

### Step 3: Add Your Name

Add your name under the Contributors section in README.md.  
Do not change anything else.

### Step 4: Commit and Push

```bash
git add README.md
git commit -m "Add my name to contributors"
git push origin main
```

### Step 5: Create a Pull Request

- Go to the original repository
- Click Pull Requests → New Pull Request
- Ensure it is from your fork → main
- Submit the pull request

### Step 6: Review & Merge

I will review your pull request.  
If everything is correct, it will be merged and closed.  
Incorrect or sloppy pull requests will be rejected.
